## 1. 📝 题目描述

- [leetcode](https://leetcode.cn/problems/add-binary/)

给你两个二进制字符串 `a` 和 `b`，以二进制字符串的形式返回它们的和。

---

示例 1：

```txt
输入：a = "11", b = "1"
输出："100"
```

---

示例 2：

```txt
输入：a = "1010", b = "1011"
输出："10101"
```

---

提示：

- `1 <= a.length, b.length <= 10^4`
- `a` 和 `b` 仅由字符 `'0'` 或 `'1'` 组成
- 字符串如果不是 `"0"`，就不含前导零

- 时间复杂度：$O(\max(m, n))$，其中 $m$ 和 $n$ 分别是字符串 `a` 和 `b` 的长度，需要遍历较长字符串的所有位
- 空间复杂度：$O(\max(m, n))$，结果字符串的长度最多为 $\max(m, n) + 1$

算法思路：

- 模拟手动计算二进制加法的过程，从两个字符串的末尾开始逐位相加
- 用变量 `carry` 记录进位，每次将对应位的数字与进位相加，得到当前位的值 `sum % 2` 和新的进位 `sum / 2`
- 当两个指针和进位都处理完毕时停止，将结果反转（或从高位向低位拼接）即为答案
