## 1. 📝 题目描述

- [leetcode](https://leetcode.cn/problems/unique-number-of-occurrences/)

给你一个整数数组 `arr`，如果每个数的出现次数都是独一无二的，就返回 `true`；否则返回 `false`。

---

- **示例 1：**

```txt
输入：arr = [1,2,2,1,1,3]
输出：true
解释：在该数组中，1 出现了 3 次，2 出现了 2 次，3 只出现了 1 次。没有两个数的出现次数相同。
```

- **示例 2：**

```txt
输入：arr = [1,2]
输出：false
```

- **示例 3：**

```txt
输入：arr = [-3,0,1,-3,1,1,1,-3,10,0]
输出：true
```

---

**提示：**

- `1 <= arr.length <= 1000`
- `-1000 <= arr[i] <= 1000`
