## 1. 📝 题目描述

- [leetcode](https://leetcode.cn/problems/decrease-elements-to-make-array-zigzag/)

给你一个整数数组 `nums`，每次 操作 会从中选择一个元素并 将该元素的值减少 1。

如果符合下列情况之一，则数组 `A` 就是 锯齿数组：

- 每个偶数索引对应的元素都大于相邻的元素，即 `A[0] > A[1] < A[2] > A[3] < A[4] > ...`
- 或者，每个奇数索引对应的元素都大于相邻的元素，即 `A[0] < A[1] > A[2] < A[3] > A[4] < ...`

返回将数组 `nums` 转换为锯齿数组所需的最小操作次数。

---

示例 1：

```txt
输入：nums = [1,2,3]
输出：2
解释：我们可以把 2 递减到 0，或把 3 递减到 1。
```

示例 2：

```txt
输入：nums = [9,6,1,6,2]
输出：4
```

---

提示：

- `1 <= nums.length <= 1000`
- `1 <= nums[i] <= 1000`

- 时间复杂度：$O(n)$，其中 $n$ 是数组的长度
- 空间复杂度：$O(1)$，只使用常数额外空间

算法思路：

- 锯齿数组有两种模式：偶数下标为谷值或奇数下标为谷值
- 分别计算两种模式所需的操作次数，取较小值
- 对于每个需要成为谷值的位置 `i`，需要将 `nums[i]` 减小到 `min(左邻居, 右邻居) - 1`，操作次数为 `max(0, nums[i] - min(neighbors) + 1)`
