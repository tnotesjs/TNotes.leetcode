## 1. 📝 题目描述

- [leetcode](https://leetcode.cn/problems/count-the-number-of-houses-at-a-certain-distance-i/)

给你三个 正整数 `n` 、`x` 和 `y`。

在城市中，存在编号从 `1` 到 `n` 的房屋，由 `n` 条街道相连。对所有 `1 <= i < n`，都存在一条街道连接编号为 `i` 的房屋与编号为 `i + 1` 的房屋。另存在一条街道连接编号为 `x` 的房屋与编号为 `y` 的房屋。

对于每个 `k`（`1 <= k <= n`），你需要找出所有满足要求的 房屋对 `[house1, house2]`，即从 `house1` 到 `house2` 需要经过的 最少 街道数为 `k`。

返回一个下标从 1 开始且长度为 `n` 的数组 `result`，其中 `result[k]` 表示所有满足要求的房屋对的数量，即从一个房屋到另一个房屋需要经过的 最少 街道数为 `k`。

注意，`x` 与 `y` 可以 相等。

## 2. 🎯 s.1 - BFS 枚举

::: code-group

:::

- 时间复杂度：$O(n^2)$，枚举所有房屋对
- 空间复杂度：$O(n)$，存储结果数组

算法思路：

- 枚举所有房屋对 (i, j)，计算三种路径的最短距离：直接路径 |i-j|、经过 x→y 的路径、经过 y→x 的路径
- 取三者最小值作为该房屋对的最短距离
- 每对 (i, j) 对结果贡献 2（正反两个方向）
