## 1. 📝 题目描述

- [leetcode](https://leetcode.cn/problems/find-if-array-can-be-sorted/)

给你一个下标从 0 开始且全是 正 整数的数组 `nums`。

一次 操作 中，如果两个 相邻 元素在二进制下 设置位 的数目 相同，那么你可以将这两个元素交换。你可以执行这个操作 任意次 （也可以 0 次）。

> set bit 指在某数的二进制表示中值为 1 的二进制位。

如果你可以使数组变为非降序，请你返回 `true`，否则返回 `false`。

---

示例 1：

```txt
输入：nums = [8,4,2,30,15]
输出：true
解释：我们先观察每个元素的二进制表示。 2，4 和 8 分别都只有一个数位为 1，分别为 "10"，"100" 和 "1000"。15 和 30 分别有 4 个数位为 1 ："1111" 和 "11110"。
我们可以通过 4 个操作使数组非降序：
- 交换 nums[0] 和 nums[1]。8 和 4 分别只有 1 个数位为 1。数组变为 [4,8,2,30,15]。
- 交换 nums[1] 和 nums[2]。8 和 2 分别只有 1 个数位为 1。数组变为 [4,2,8,30,15]。
- 交换 nums[0] 和 nums[1]。4 和 2 分别只有 1 个数位为 1。数组变为 [2,4,8,30,15]。
- 交换 nums[3] 和 nums[4]。30 和 15 分别有 4 个数位为 1，数组变为 [2,4,8,15,30]。
数组变成有序的，所以我们返回 true。
注意我们还可以通过其他的操作序列使数组变得有序。
```

示例 2：

```txt
输入：nums = [1,2,3,4,5]
输出：true
解释：数组已经是非降序的，所以我们返回 true。
```

示例 3：

```txt
输入：nums = [3,16,8,4,2]
输出：false
解释：无法通过操作使数组变为非降序。
```

---

提示：

- `1 <= nums.length <= 100`
- `1 <= nums[i] <= 2^8`

- 时间复杂度：$O(N)$，其中 N 为数组长度
- 空间复杂度：$O(1)$，只使用了常数级别的额外空间

算法思路：

- 相同 set bit 数量的相邻元素可以任意交换，等价于同一组内可以自由排序
- 将数组按相邻元素的 set bit 数量分组，记录每组的最小值和最大值
- 遍历时检查前一组的最大值是否大于当前组的最小值，若是则无法排序，返回 `false`
