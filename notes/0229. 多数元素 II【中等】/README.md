## 1. 📝 题目描述

- [leetcode](https://leetcode.cn/problems/majority-element-ii/)

给定一个大小为 _n_ 的整数数组，找出其中所有出现超过 `⌊ n/3 ⌋` 次的元素。

**示例 1：**

```txt
输入：nums = [3,2,3]
输出：[3]
```

- **示例 2：**

```txt
输入：nums = [1]
输出：[1]
```

- **示例 3：**

```txt
输入：nums = [1,2]
输出：[1,2]
```

---

**提示：**

- `1 <= nums.length <= 5 * 10^4`
- `-10^9 <= nums[i] <= 10^9`

**进阶：** 尝试设计时间复杂度为 O(n)、空间复杂度为 O(1)的算法解决此问题。
