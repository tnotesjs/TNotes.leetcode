## 1. 📝 题目描述

- [leetcode](https://leetcode.cn/problems/count-sorted-vowel-strings/)

给你一个整数 `n`，请返回长度为 `n` 、仅由元音 (`a`, `e`, `i`, `o`, `u`) 组成且按 字典序排列 的字符串数量。

字符串 `s` 按 字典序排列 需要满足：对于所有有效的 `i`，`s[i]` 在字母表中的位置总是与 `s[i+1]` 相同或在 `s[i+1]` 之前。

---

示例 1：

```txt
输入：n = 1
输出：5

解释：
仅由元音组成的 5 个字典序字符串为 ["a","e","i","o","u"]
```

示例 2：

```txt
输入：n = 2
输出：15

解释：
仅由元音组成的 15 个字典序字符串为
["aa","ae","ai","ao","au","ee","ei","eo","eu","ii","io","iu","oo","ou","uu"]
注意，"ea" 不是符合题意的字符串，因为 'e' 在字母表中的位置比 'a' 靠后
```

示例 3：

```txt
输入：n = 33
输出：66045
```

---

提示：

- `1 <= n <= 50`

- 时间复杂度：$O(5n)$，其中 $n$ 是字符串长度
- 空间复杂度：$O(1)$，只用 5 个变量

算法思路：

- dp[j] 表示以第 j 个元音字母结尾的长度为 i 的字符串数
- 字符串必须字典序非递减，因此 dp[j] = sum(dp[0..j])
- 从长度 1 迭代到 n，最终答案为 sum(dp)
