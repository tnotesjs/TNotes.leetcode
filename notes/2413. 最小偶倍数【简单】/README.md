## 1. 📝 题目描述

- [leetcode](https://leetcode.cn/problems/smallest-even-multiple/)

给你一个正整数 `n`，返回 `2` 和 `n` 的最小公倍数（正整数）。

---

示例 1：

```txt
输入：n = 5
输出：10

解释：
5 和 2 的最小公倍数是 10。
```

---

示例 2：

```txt
输入：n = 6
输出：6

解释：
6 和 2 的最小公倍数是 6。
注意数字会是它自身的倍数。
```

---

提示：

- `1 <= n <= 150`

- 时间复杂度：$O(1)$，只需要判断奇偶性
- 空间复杂度：$O(1)$，只使用常数额外空间

算法思路：

- 如果 n 是偶数，n 本身就是 2 的倍数，所以 2 和 n 的最小公倍数就是 n
- 如果 n 是奇数，n 和 2 互质，最小公倍数为 n × 2
- 直接判断 n 的奇偶性即可
