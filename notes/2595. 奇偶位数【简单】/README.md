## 1. 📝 题目描述

- [leetcode](https://leetcode.cn/problems/number-of-even-and-odd-bits/)

给你一个 正 整数 `n`。

用 `even` 表示在 `n` 的二进制形式（下标从 0 开始）中值为 `1` 的偶数下标的个数。

用 `odd` 表示在 `n` 的二进制形式（下标从 0 开始）中值为 `1` 的奇数下标的个数。

请注意，在数字的二进制表示中，位下标的顺序从右到左。

返回整数数组 `answer`，其中 `answer = [even, odd]`。

---

示例 1：

```txt
输入：n = 50
输出：[1,2]

解释：
50 的二进制表示是 `110010`。
在下标 1，4，5 对应的值为 1。
```

---

示例 2：

```txt
输入：n = 2
输出：[0,1]

解释：
2 的二进制表示是 `10`。
只有下标 1 对应的值为 1。
```

---

提示：

- `1 <= n <= 1000`

- 时间复杂度：$O(\log N)$，其中 N 是数字的值，需要遍历二进制的每一位
- 空间复杂度：$O(1)$，只使用常数额外空间

算法思路：

- 使用位运算遍历数字的每一位二进制位（从右到左，索引从 0 开始）
- 通过 `n & 1` 检查当前最低位是否为 1
- 根据当前索引的奇偶性，分别累加 even 或 odd 计数器
- 使用右移运算 `n >>= 1` 处理下一位，同时索引加 1
- 返回 [even, odd] 数组
