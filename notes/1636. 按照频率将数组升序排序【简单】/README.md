## 1. 📝 题目描述

- [leetcode](https://leetcode.cn/problems/sort-array-by-increasing-frequency/)

给你一个整数数组 `nums`，请你将数组按照每个值的频率升序排序。如果有多个值的频率相同，请你按照数值本身将它们降序排序。

请你返回排序后的数组。

---

示例 1：

```txt
输入：nums = [1,1,2,2,2,3]
输出：[3,1,1,2,2,2]
```

解释：'3' 频率为 1，'1' 频率为 2，'2' 频率为 3。

---

示例 2：

```txt
输入：nums = [2,3,1,3,2]
输出：[1,3,3,2,2]
```

解释：'2' 和 '3' 频率都为 2，所以它们之间按照数值本身降序排序。

---

示例 3：

```txt
输入：nums = [-1,1,-6,4,5,-6,1,4,1]
输出：[5,-1,4,4,-6,-6,1,1,1]
```

---

提示：

- `1 <= nums.length <= 100`
- `-100 <= nums[i] <= 100`

- 时间复杂度：$O(N \log N)$，其中 N 是数组 nums 的长度，统计频率需要 O(N)，排序需要 $O(N \log N)$
- 空间复杂度：$O(N)$，哈希表存储频率信息

算法思路：

- 创建哈希表 `freq`，遍历数组 `nums`，统计每个数字的出现频率
- 使用自定义比较函数对 `nums` 进行原地排序：
  - 获取两个数字 `a` 和 `b` 的频率 `fa` 和 `fb`
  - 如果频率不同，按频率升序排列（`fa - fb`）
  - 如果频率相同，按数值降序排列（`b - a`）
- 返回排序后的数组
