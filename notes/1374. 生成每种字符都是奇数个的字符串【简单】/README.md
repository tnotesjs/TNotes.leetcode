## 1. 📝 题目描述

- [leetcode](https://leetcode.cn/problems/generate-a-string-with-characters-that-have-odd-counts/)

给你一个整数 `n`，请你返回一个含 `n` 个字符的字符串，其中每种字符在该字符串中都恰好出现奇数次。

返回的字符串必须只含小写英文字母。如果存在多个满足题目要求的字符串，则返回其中任意一个即可。

---

示例 1：

```txt
输入：n = 4
输出："pppz"

解释：
"pppz" 是一个满足题目要求的字符串，因为 'p' 出现 3 次，且 'z' 出现 1 次。
当然，还有很多其他字符串也满足题目要求，比如："ohhh" 和 "love"。
```

---

示例 2：

```txt
输入：n = 2
输出："xy"

解释：
"xy" 是一个满足题目要求的字符串，因为 'x' 和 'y' 各出现 1 次。
当然，还有很多其他字符串也满足题目要求，比如："ag" 和 "ur"。
```

---

示例 3：

```txt
输入：n = 7
输出："holasss"
```

---

提示：

- `1 <= n <= 500`

- 时间复杂度：$O(n)$，生成长度为 $n$ 的字符串
- 空间复杂度：$O(n)$，结果字符串占用

算法思路：

- 若 $n$ 为奇数，直接返回 `a` 重复 $n$ 次
- 若 $n$ 为偶数，前 $n-1$ 个用 `a`，末尾加一个不同字符（如 `b`）保证两种字符均为奇数次
