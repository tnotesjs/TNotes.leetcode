## 1. 📝 题目描述

- [leetcode](https://leetcode.cn/problems/reordered-power-of-2/)

给定正整数  `n` ，我们按任何顺序（包括原始顺序）将数字重新排序，注意其前导数字不能为零。

如果我们可以通过上述方式得到  2 的幂，返回 `true`；否则，返回 `false`。

---

- **示例 1：**

```txt
输入：n = 1
输出：true
```

- **示例 2：**

```txt
输入：n = 10
输出：false
```

---

**提示：**

- `1 <= n <= 10^9`
