## 1. 📝 题目描述

- [leetcode](https://leetcode.cn/problems/reorganize-string/)

给定一个字符串 `s` ，检查是否能重新排布其中的字母，使得两相邻的字符不同。

返回 `s` 的任意可能的重新排列。若不可行，返回空字符串 `""` 。

示例 1：

```txt
输入: s = "aab"
输出: "aba"
```

- 示例 2：

```txt
输入: s = "aaab"
输出: ""
```

---

提示：

- `1 <= s.length <= 500`
- `s` 只包含小写字母
