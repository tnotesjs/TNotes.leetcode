## 1. 题目描述

- [leetcode](https://leetcode.cn/problems/single-number-ii/)

给你一个整数数组 `nums`，除某个元素仅出现一次外，其余每个元素都恰出现三次。请你找出并返回那个只出现了一次的元素。

你必须设计并实现线性时间复杂度的算法且使用常数级空间来解决此问题。

---

示例 1：

```txt
输入：nums = [2, 2, 3, 2]
输出：3
```

---

示例 2：

```txt
输入：nums = [0, 1, 0, 1, 0, 1, 99]
输出：99
```

---

提示：

- `1 <= nums.length <= 3 * 10^4`
- `-2^31 <= nums[i] <= 2^31 - 1`
- `nums` 中，除某个元素仅出现一次外，其余每个元素都恰出现三次

- 时间复杂度：$O(n)$，遍历数组一次
- 空间复杂度：$O(1)$，只使用了两个变量

算法思路：

- 用 `ones` 和 `twos` 两个变量模拟「三进制」计数器，每位分别记录出现 1 次和 2 次的状态：
  - `ones = (ones ^ num) & ~twos`：新数据进入 `ones`，但在 `twos` 中已置位的位不能再进入
  - `twos = (twos ^ num) & ~ones`：新数据进入 `twos`，但在 `ones` 中已置位的位不能再进入
- 最终 `ones` 中的每一位就是只出现在模 3 余 1 的位，也就是那个唯一数的每一位

- 时间复杂度：$O(n)$，外层循环 32 次，内层循环 n 次，总体仍是线性时间
- 空间复杂度：$O(1)$，只使用常数额外空间

算法思路：

- 对 32 位整型的每一位，统计数组中所有数在该位上为 1 的次数
- 由于除答案外每个数都出现 3 次，如果某位的计数 `count % 3 != 0`，说明答案在该位上是 1
- 用 `result |= (1 << i)` 将每位结果拼回最终答案
