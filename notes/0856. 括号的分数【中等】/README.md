## 1. 📝 题目描述

- [leetcode](https://leetcode.cn/problems/score-of-parentheses/)

给定一个平衡括号字符串 `S`，按下述规则计算该字符串的分数：

- `()` 得 1 分。
- `AB` 得 `A + B` 分，其中 A 和 B 是平衡括号字符串。
- `(A)` 得 `2 * A` 分，其中 A 是平衡括号字符串。

---

示例 1：

```txt
输入："()"
输出：1
```

示例 2：

```txt
输入："(())"
输出：2
```

示例 3：

```txt
输入："()()"
输出：2
```

示例 4：

```txt
输入："(()(()))"
输出：6
```

---

提示：

1.  `S` 是平衡括号字符串，且只含有 `(` 和 `)`。
2.  `2 <= S.length <= 50`

- 时间复杂度：$O(n)$，其中 n 是字符串长度
- 空间复杂度：$O(n)$

算法思路：

- 用栈维护当前层的分数，遇 `(` 压入 0，遇 `)` 弹出并计算
- `()` 计 1 分，`(A)` 计 `2*A` 分，累加到父层
