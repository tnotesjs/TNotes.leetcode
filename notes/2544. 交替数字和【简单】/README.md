## 1. 📝 题目描述

- [leetcode](https://leetcode.cn/problems/alternating-digit-sum/)

给你一个正整数 `n`。`n` 中的每一位数字都会按下述规则分配一个符号：

- 最高有效位上的数字分配到正号。
- 剩余每位上数字的符号都与其相邻数字相反。

返回所有数字及其对应符号的和。

---

示例 1：

```txt
输入：n = 521
输出：4
解释：(+5) + (-2) + (+1) = 4
```

---

示例 2：

```txt
输入：n = 111
输出：1
解释：(+1) + (-1) + (+1) = 1
```

---

示例 3：

```txt
输入：n = 886996
输出：0
解释：(+8) + (-8) + (+6) + (-9) + (+9) + (-6) = 0
```

---

提示：

- `1 <= n <= 10^9`

- 时间复杂度：$O(\log N)$，其中 N 是数字的值，数字有 $\log N$ 位
- 空间复杂度：$O(\log N)$，字符串转换所需空间

算法思路：

- 将数字转换为字符串，从高位到低位遍历每一位
- 根据位置索引的奇偶性决定符号：偶数索引（从 0 开始）为正号，奇数索引为负号
- 将每一位数字乘以对应的符号后累加到总和
- 返回最终的交替数字和
