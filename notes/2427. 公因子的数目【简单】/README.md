## 1. 📝 题目描述

- [leetcode](https://leetcode.cn/problems/number-of-common-factors/)

给你两个正整数 `a` 和 `b`，返回 `a` 和 `b` 的公因子的数目。

如果 `x` 可以同时整除 `a` 和 `b`，则认为 `x` 是 `a` 和 `b` 的一个公因子。

---

示例 1：

```txt
输入：a = 12, b = 6
输出：4
解释：12 和 6 的公因子是 1、2、3、6。
```

---

示例 2：

```txt
输入：a = 25, b = 30
输出：2
解释：25 和 30 的公因子是 1、5。
```

---

提示：

- `1 <= a, b <= 1000`

- 空间复杂度：$O(1)$，只使用常数额外空间

- 对于每个数 i，检查是否同时能整除 a 和 b
