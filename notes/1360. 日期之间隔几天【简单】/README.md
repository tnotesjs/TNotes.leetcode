## 1. 📝 题目描述

- [leetcode](https://leetcode.cn/problems/number-of-days-between-two-dates/)

请你编写一个程序来计算两个日期之间隔了多少天。

日期以字符串形式给出，格式为 `YYYY-MM-DD`，如示例所示。

---

示例 1：

```txt
输入：date1 = "2019-06-29", date2 = "2019-06-30"
输出：1
```

---

示例 2：

```txt
输入：date1 = "2020-01-15", date2 = "2019-12-31"
输出：15
```

---

提示：

- 给定的日期是 `1971` 年到 `2100` 年之间的有效日期。

- 时间复杂度：$O(Y)$，逐年累加至目标年份，$Y\le130$
- 空间复杂度：$O(1)$

算法思路：

- 将日期解析为年月日，计算自 1971-01-01 起累计的天数（逐年加 365/366，再加上当年的月日偏移）
- 对两个日期分别计算天数后取差的绝对值即答案
