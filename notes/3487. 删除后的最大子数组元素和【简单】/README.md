## 1. 📝 题目描述

- [leetcode](https://leetcode.cn/problems/maximum-unique-subarray-sum-after-deletion/)

给你一个整数数组 `nums`。

你可以从数组 `nums` 中删除任意数量的元素，但不能将其变为空数组。执行删除操作后，选出 `nums` 中满足下述条件的一个子数组：

1. 子数组中的所有元素互不相同。
2. 最大化子数组的元素和。

返回子数组的最大元素和。

子数组是数组的一个连续、非空的元素序列。

---

示例 1：

```txt
输入：nums = [1,2,3,4,5]
输出：15
```

解释：

不删除任何元素，选中整个数组得到最大元素和。

---

示例 2：

```txt
输入：nums = [1,1,0,1,1]
输出：1
```

解释：

删除元素 `nums[0] == 1`、`nums[1] == 1`、`nums[2] == 0` 和 `nums[3] == 1`。选中整个数组 `[1]` 得到最大元素和。

---

示例 3：

```txt
输入：nums = [1,2,-1,-2,1,0,-1]
输出：3
```

解释：

删除元素 `nums[2] == -1` 和 `nums[3] == -2` ，从 `[1, 2, 1, 0, -1]` 中选中子数组 `[2, 1]` 以获得最大元素和。

---

提示：

- `1 <= nums.length <= 100`
- `-100 <= nums[i] <= 100`

- 时间复杂度：$O(n)$，n 是数组 nums 的长度
- 空间复杂度：$O(n)$，哈希集合最多存储 n 个不同的正数

算法思路：

- 关键观察：由于可以删除任意元素，我们实际上可以选择保留哪些元素形成子序列
- 为了使子数组元素互不相同且和最大，我们应该保留所有不同的正数
- 对于负数和 0，不保留它们可以使和更大
- 如果数组中没有正数，则返回数组中的最大值（必须保留至少一个元素）
