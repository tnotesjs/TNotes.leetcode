## 1. 题目描述

- [leetcode](https://leetcode.cn/problems/unique-binary-search-trees/)

给你一个整数 `n`，求恰由 `n` 个节点组成且节点值从 `1` 到 `n` 互不相同的二叉搜索树有多少种？返回满足题意的二叉搜索树的种数。

---

示例 1：

![img](https://cdn.jsdelivr.net/gh/tnotesjs/imgs@main/2025-09-10-17-51-42.png)

```txt
输入：n = 3
输出：5
```

示例 2：

```txt
输入：n = 1
输出：1
```

---

提示：

- `1 <= n <= 19`

- 时间复杂度：$O(n^2)$，双层循环填表
- 空间复杂度：$O(n)$，`dp` 数组的大小

算法思路：

- `dp[i]` 表示 `i` 个节点能组成的不同 BST 的数量，即卡特兰数 $C_n$
- 枚举每个节点 `j` 作为根，左子树有 `j-1` 个节点，右子树有 `i-j` 个节点
- 转移方程：$dp[i] = \sum_{j=1}^{i} dp[j-1] \times dp[i-j]$
- 初始化：$dp[0] = dp[1] = 1$，空树和单节点树各有 1 种

## 3. 空树本身也属于二叉搜索树（BST）吗？【AI】

是的，在算法题里通常认为空树也是一棵合法的 BST。

因为它没有任何节点，所以不会违反：

```txt
左子树 < 根节点 < 右子树
```

这个性质。

所以在这题里：

```js
dp[0] = 1
```

表示：空树算 1 种合法情况，主要是为了方便组合左右子树。

## 4. 卡特兰数是什么？【AI】

卡特兰数是一类常见计数序列，用来统计很多“递归划分结构”的数量，比如：

```txt
不同的二叉搜索树数量
合法括号序列数量
不同的二叉树形态数量
多边形三角剖分数量
```

它的递推形式是：

```txt
C₀ = 1
Cₙ = C₀Cₙ₋₁ + C₁Cₙ₋₂ + ... + Cₙ₋₁C₀
```

对应到这题就是：

```js
dp[i] += dp[j - 1] * dp[i - j]
```

也就是：

```txt
左子树种数 × 右子树种数
```

前几个卡特兰数是：

```txt
C₀ = 1
C₁ = 1
C₂ = 2
C₃ = 5
C₄ = 14
C₅ = 42
```

所以当 `n = 3` 时，不同 BST 的数量是：

```txt
C₃ = 5
```

一句话总结：卡特兰数就是专门用来数这类“左右两边递归组合”的结构数量的数列。

::: tip 备注：“卡特兰数列”和“卡特兰数”

下面这个递推关系得到的数列，就是卡特兰数列。

```txt
C₀ = 1
Cₙ = C₀Cₙ₋₁ + C₁Cₙ₋₂ + ... + Cₙ₋₁C₀
```

卡特兰数列的前几项是：1, 1, 2, 5, 14, 42, 132, ...

在这道 BST 题里：`dp[n] = Cₙ`，也就是说，`n` 个节点能组成的不同二叉搜索树数量，正好是第 `n` 个卡特兰数。

:::
