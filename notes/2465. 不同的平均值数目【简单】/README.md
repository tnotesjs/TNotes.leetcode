## 1. 📝 题目描述

- [leetcode](https://leetcode.cn/problems/number-of-distinct-averages/)

给你一个下标从 0 开始长度为偶数的整数数组 `nums`。

只要 `nums` 不是空数组，你就重复执行以下步骤：

- 找到 `nums` 中的最小值，并删除它。
- 找到 `nums` 中的最大值，并删除它。
- 计算删除两数的平均值。

两数 `a` 和 `b` 的平均值为 `(a + b) / 2`。

- 比方说，`2` 和 `3` 的平均值是 `(2 + 3) / 2 = 2.5`。

返回上述过程能得到的不同平均值的数目。

注意，如果最小值或者最大值有重复元素，可以删除任意一个。

---

示例 1：

```txt
输入：nums = [4,1,4,0,3,5]
输出：2

解释：
1. 删除 0 和 5，平均值是 (0 + 5) / 2 = 2.5，现在 nums = [4,1,4,3]。
2. 删除 1 和 4，平均值是 (1 + 4) / 2 = 2.5，现在 nums = [4,3]。
3. 删除 3 和 4，平均值是 (3 + 4) / 2 = 3.5。
2.5，2.5 和 3.5 之中总共有 2 个不同的数，我们返回 2。
```

---

示例 2：

```txt
输入：nums = [1,100]
输出：1

解释：
删除 1 和 100 后只有一个平均值，所以我们返回 1。
```

---

提示：

- `2 <= nums.length <= 100`
- `nums.length` 是偶数。
- `0 <= nums[i] <= 100`

- 时间复杂度：$O(N \log N)$，其中 N 是数组 nums 的长度，主要消耗在排序
- 空间复杂度：$O(N)$，哈希集合最多存储 N/2 个不同的和

算法思路：

- 对数组排序后，最小值在左端，最大值在右端
- 使用双指针从两端向中间移动，每次取最小值和最大值
- 计算两数之和并存入哈希集合（两数之和相同则平均值相同，避免浮点数精度问题）
- 返回集合大小即为不同平均值的数目
