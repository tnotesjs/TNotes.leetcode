## 1. 📝 题目描述

- [leetcode](https://leetcode.cn/problems/find-the-longest-balanced-substring-of-a-binary-string/)

给你一个仅由 `0` 和 `1` 组成的二进制字符串 `s`。

如果子字符串中 所有的 `0` 都在 `1` 之前 且其中 `0` 的数量等于 `1` 的数量，则认为 `s` 的这个子字符串是平衡子字符串。请注意，空子字符串也视作平衡子字符串。

返回 `s` 中最长的平衡子字符串长度。

子字符串是字符串中的一个连续字符序列。

---

示例 1：

```txt
输入：s = "01000111"
输出：6

解释：
最长的平衡子字符串是 "000111"，长度为 6。
```

---

示例 2：

```txt
输入：s = "00111"
输出：4

解释：
最长的平衡子字符串是 "0011"，长度为  4。
```

---

示例 3：

```txt
输入：s = "111"
输出：0

解释：
除了空子字符串之外不存在其他平衡子字符串，所以答案为 0。
```

---

提示：

- `1 <= s.length <= 50`
- `'0' <= s[i] <= '1'`

- 时间复杂度：$O(N)$，其中 N 是字符串长度
- 空间复杂度：$O(1)$，只使用常数额外空间

算法思路：

- 遍历字符串，维护当前连续 0 和连续 1 的数量
- 遇到 0 时：如果前面有 1（oneCount > 0），说明开始了新的一段，重置计数器；否则继续累加 zeroCount
- 遇到 1 时：累加 oneCount，然后计算平衡子字符串长度：取 zeroCount 和 oneCount 的最小值再乘2
- 更新最大长度并返回
