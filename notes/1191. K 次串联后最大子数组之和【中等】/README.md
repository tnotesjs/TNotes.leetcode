## 1. 📝 题目描述

- [leetcode](https://leetcode.cn/problems/k-concatenation-maximum-sum/)

给定一个整数数组 `arr` 和一个整数 `k` ，通过重复 `k` 次来修改数组。

例如，如果 `arr = [1, 2]` ， `k = 3` ，那么修改后的数组将是 `[1, 2, 1, 2, 1, 2]` 。

返回修改后的数组中的最大的子数组之和。注意，子数组长度可以是 `0`，在这种情况下它的总和也是 `0`。

由于 结果可能会很大，需要返回的 `10^9 + 7` 的 模 。

---

- 示例 1：

```txt
输入：arr = [1,2], k = 3
输出：9
```

- 示例 2：

```txt
输入：arr = [1,-2,1], k = 5
输出：2
```

- 示例 3：

```txt
输入：arr = [-1,-2], k = 7
输出：0
```

---

提示：

- `1 <= arr.length <= 10^5`
- `1 <= k <= 10^5`
- `-10^4 <= arr[i] <= 10^4`
