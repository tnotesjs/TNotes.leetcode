## 1. 📝 题目描述

- [leetcode](https://leetcode.cn/problems/squares-of-a-sorted-array)

给你一个按非递减顺序排序的整数数组 `nums`，返回每个数字的平方组成的新数组，要求也按非递减顺序排序。

---

示例 1：

```txt
输入：nums = [-4, -1, 0, 3, 10]
输出：[0, 1, 9, 16, 100]

解释：
平方后，数组变为 [16, 1, 0, 9, 100]
排序后，数组变为 [0, 1, 9, 16, 100]
```

---

示例 2：

```txt
输入：nums = [-7, -3, 2, 3, 11]
输出：[4, 9, 9, 49, 121]
```

---

提示：

- `1 <= nums.length <= 10^4`
- `-10^4 <= nums[i] <= 10^4`
- `nums` 已按非递减顺序排序

进阶：

- 请你设计时间复杂度为 `O(n)` 的算法解决本问题
