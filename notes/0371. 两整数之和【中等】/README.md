## 1. 📝 题目描述

- [leetcode](https://leetcode.cn/problems/sum-of-two-integers/)

给你两个整数 `a` 和 `b` ，**不使用** 运算符  `+` 和  `-` ​​​​​​​，计算并返回两整数之和。

---

- **示例 1：**

```txt
输入：a = 1, b = 2
输出：3
```

- **示例 2：**

```txt
输入：a = 2, b = 3
输出：5
```

---

**提示：**

- `-1000 <= a, b <= 1000`
