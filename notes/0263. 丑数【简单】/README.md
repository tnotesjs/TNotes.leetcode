## 1. 📝 题目描述

- [leetcode](https://leetcode.cn/problems/ugly-number/)

**丑数** 就是只包含质因数  `2`、`3` 和 `5`  的  *正*  整数。

给你一个整数 `n` ，请你判断 `n` 是否为 **丑数** 。如果是，返回 `true` ；否则，返回 `false` 。

---

- **示例 1：**

```txt
输入：n = 6
输出：true
解释：6 = 2 × 3
```

- **示例 2：**

```txt
输入：n = 1
输出：true
解释：1 没有质因数。
```

- **示例 3：**

```txt
输入：n = 14
输出：false
解释：14 不是丑数，因为它包含了另外一个质因数 7 。
```

---

**提示：**

- `-2^31 <= n <= 2^31 - 1`
