## 1. 📝 题目描述

- [leetcode](https://leetcode.cn/problems/count-numbers-with-unique-digits/)

给你一个整数 `n` ，统计并返回各位数字都不同的数字 `x` 的个数，其中 `0 <= x < 10^n` 。

---

- **示例 1：**

```txt
输入：n = 2
输出：91
解释：答案应为除去 11、22、33、44、55、66、77、88、99 外，在 0 ≤ x < 100 范围内的所有数字。
```

- **示例 2：**

```txt
输入：n = 0
输出：1
```

---

**提示：**

- `0 <= n <= 8`
