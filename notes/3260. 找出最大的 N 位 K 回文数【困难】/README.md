## 1. 📝 题目描述

- [leetcode](https://leetcode.cn/problems/find-the-largest-palindrome-divisible-by-k/)

给你两个 正整数 `n` 和 `k`。

如果整数 `x` 满足以下全部条件，则该整数是一个 k 回文数：

- `x` 是一个 回文数。
- `x` 可以被 `k` 整除。

> - 回文数
> - 如果一个整数向前和向后读都相同，则它是一个 回文数。 例如，121 是回文数，而 123 不是。

以字符串形式返回 最大的 `n` 位 k 回文数。

注意，该整数 不 含前导零。

---

示例 1：

输入： n = 3, k = 5

输出： "595"

解释：

595 是最大的 3 位 k 回文数。

---

示例 2：

输入： n = 1, k = 4

输出： "8"

解释：

1 位 k 回文数只有 4 和 8。

---

示例 3：

输入： n = 5, k = 6

输出： "89898"

---

提示：

- `1 <= n <= 10^5`
- `1 <= k <= 9`
