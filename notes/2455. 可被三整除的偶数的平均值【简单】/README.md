## 1. 📝 题目描述

- [leetcode](https://leetcode.cn/problems/average-value-of-even-numbers-that-are-divisible-by-three/)

给你一个由正整数组成的整数数组 `nums`，返回其中可被 `3` 整除的所有偶数的平均值。

注意：`n` 个元素的平均值等于 `n` 个元素求和再除以 `n`，结果向下取整到最接近的整数。

---

示例 1：

```txt
输入：nums = [1,3,6,10,12,15]
输出：9

解释：
6 和 12 是可以被 3 整除的偶数。(6 + 12) / 2 = 9。
```

---

示例 2：

```txt
输入：nums = [1,2,4,7,10]
输出：0

解释：
不存在满足题目要求的整数，所以返回 0。
```

---

提示：

- `1 <= nums.length <= 1000`
- `1 <= nums[i] <= 1000`

- 时间复杂度：$O(N)$，其中 N 是数组长度，需要遍历一次数组
- 空间复杂度：$O(1)$，只使用常数额外空间

算法思路：

题目要求找出可被 3 整除的偶数并计算平均值。关键观察：可被 3 整除的偶数等价于可被 6 整除的数。

- 遍历数组，筛选出满足 num % 6 === 0 的元素
- 使用 sum 累加这些元素的和，count 记录数量
- 如果 count 为 0，返回 0；否则返回 Math.floor(sum / count)
- 优化点：直接用模 6 判断，因为既是偶数又是 3 的倍数等价于是 6 的倍数
