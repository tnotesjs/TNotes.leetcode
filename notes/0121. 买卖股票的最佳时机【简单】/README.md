## 1. 📝 题目描述

- [leetcode](https://leetcode.cn/problems/best-time-to-buy-and-sell-stock)

给定一个数组 `prices`，它的第 `i` 个元素 `prices[i]` 表示一支给定股票第 `i` 天的价格。

你只能选择某一天买入这只股票，并选择在未来的某一个不同的日子卖出该股票。设计一个算法来计算你所能获取的最大利润。

返回你可以从这笔交易中获取的最大利润。如果你不能获取任何利润，返回 `0`。

---

示例 1：

```
输入：[7, 1, 5, 3, 6, 4]
输出：5
```

解释：

- 在第 2 天（股票价格 = 1）的时候买入，在第 5 天（股票价格 = 6）的时候卖出，最大利润 = 6 - 1 = 5。
- 注意利润不能是 7 - 1 = 6, 因为卖出价格需要大于买入价格；同时，你不能在买入前卖出股票。

---

示例 2：

```
输入：prices = [7, 6, 4, 3, 1]
输出：0
```

解释：在这种情况下，没有交易完成，所以最大利润为 0。

---

提示：

- `1 <= prices.length <= 10^5`
- `0 <= prices[i] <= 10^4`

- 时间复杂度：$O(n)$，其中 $n$ 是数组长度，只需遍历一次
- 空间复杂度：$O(1)$，只使用常数级别的额外空间

算法思路：

- 题目要求只能买卖一次，最大利润等于「卖出价格 - 买入价格」的最大值
- 遍历过程中维护两个变量：`minPrice` 记录遍历到当前位置的最低价格，`maxProfit` 记录当前能获得的最大利润
- 每遇到一个新的价格，用它与最低价格的差来尝试刷新最大利润，同时用这个价格来更新最低价格
