## 1. 📝 题目描述

- [leetcode](https://leetcode.cn/problems/maximum-population-year/)

给你一个二维整数数组 `logs`，其中每个 `logs[i] = [birthi, deathi]` 表示第 `i` 个人的出生和死亡年份。

年份 `x` 的人口定义为这一年期间活着的人的数目。第 `i` 个人被计入年份 `x` 的人口需要满足：`x` 在闭区间 `[birthi, deathi - 1]` 内。注意，人不应当计入他们死亡当年的人口中。

返回人口最多且最早的年份。

---

示例 1：

```txt
输入：logs = [[1993,1999],[2000,2010]]
输出：1993

解释：
人口最多为 1，而 1993 是人口为 1 的最早年份。
```

---

示例 2：

```txt
输入：logs = [[1950,1961],[1960,1971],[1970,1981]]
输出：1960

解释：
人口最多为 2，分别出现在 1960 和 1970。
其中最早年份是 1960。
```

---

提示：

- `1 <= logs.length <= 100`
- `1950 <= birthi < deathi <= 2050`

- 时间复杂度：$O(N + C)$，其中 N 是 logs 数组长度，C 是年份范围（1950-2050，固定为 101），构建差分数组需要 $O(N)$，扫描年份需要 $O(C)$
- 空间复杂度：$O(C)$，需要长度为 102 的差分数组

算法思路：

- 创建长度为 102 的差分数组 diff，映射年份 1950-2050
- 遍历 logs，对每个人的出生年份 `diff[birth-1950] += 1`，死亡年份 `diff[death-1950] -= 1`
- 通过前缀和还原每年的实际人口数
- 遍历年份 1950-2050，记录人口最多的年份（若人口相同取最早年份）
