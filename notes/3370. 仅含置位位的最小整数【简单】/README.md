## 1. 📝 题目描述

- [leetcode](https://leetcode.cn/problems/smallest-number-with-all-set-bits/)

给你一个正整数 `n`。

返回 大于等于 `n` 且二进制表示仅包含 置位 位的 最小 整数 `x`。

置位 位指的是二进制表示中值为 `1` 的位。

---

示例 1：

输入： n = 5

输出： 7

解释：

7 的二进制表示是 `"111"`。

---

示例 2：

输入： n = 10

输出： 15

解释：

15 的二进制表示是 `"1111"`。

---

示例 3：

输入： n = 3

输出： 3

解释：

3 的二进制表示是 `"11"`。

---

提示：

- `1 <= n <= 1000`

- 时间复杂度：$O(\log n)$，每次循环位数加 1
- 空间复杂度：$O(1)$，只使用了常数级别的额外空间

解题思路：

- 仅含置位位的数即二进制全为 1 的数：1, 3, 7, 15, 31, ...（即 $2^k - 1$）
- 从 1 开始，每次将 x 变为 `x * 2 + 1`，相当于二进制左移一位并在末尾加 1
- 找到第一个大于等于 n 的全 1 数即为答案
