## 1. 📝 题目描述

- [leetcode](https://leetcode.cn/problems/thousand-separator/)

给你一个整数 `n`，请你每隔三位添加点（即 "." 符号）作为千位分隔符，并将结果以字符串格式返回。

---

示例 1：

```txt
输入：n = 987
输出："987"
```

示例 2：

```txt
输入：n = 1234
输出："1.234"
```

示例 3：

```txt
输入：n = 123456789
输出："123.456.789"
```

示例 4：

```txt
输入：n = 0
输出："0"
```

---

提示：

- `0 <= n < 2^31`

算法思路：
