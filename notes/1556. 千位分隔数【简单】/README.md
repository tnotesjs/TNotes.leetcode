## 1. 📝 题目描述

- [leetcode](https://leetcode.cn/problems/thousand-separator/)

给你一个整数 `n`，请你每隔三位添加点（即 "." 符号）作为千位分隔符，并将结果以字符串格式返回。

---

示例 1：

```txt
输入：n = 987
输出："987"
```

---

示例 2：

```txt
输入：n = 1234
输出："1.234"
```

---

示例 3：

```txt
输入：n = 123456789
输出："123.456.789"
```

---

示例 4：

```txt
输入：n = 0
输出："0"
```

---

提示：

- `0 <= n < 2^31`

:::

- 时间复杂度：$O(L)$，L 为数字的位数
- 空间复杂度：$O(L)$，保存结果字符串

算法思路：

- 使用正则表达式 `/\B(?=(\d{3})+$)/g` 在非单词边界且后面跟随 3 的倍数个数字的位置插入点
- 正则表达式 `/\B(?=(\d{3})+$)/g` 解析：
  - `\B`：匹配非单词边界位置（避免在字符串开头插入分隔符）
  - `(?=(\d{3})+$)`：正向肯定预查，匹配后面满足条件的位置
    - `(\d{3})+`：一个或多个连续的 3 位数字
    - `$`：字符串结尾
  - `/g`：全局匹配
