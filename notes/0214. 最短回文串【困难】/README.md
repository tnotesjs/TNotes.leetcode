## 1. 📝 题目描述

- [leetcode](https://leetcode.cn/problems/shortest-palindrome/)

给定一个字符串 `s`，你可以通过在字符串前面添加字符将其转换为回文串。找到并返回可以用这种方式转换的最短回文串。

> 回文 串是向前和向后读都相同的字符串。

---

- **示例 1：**

```txt
输入：s = "aacecaaa"
输出："aaacecaaa"
```

- **示例 2：**

```txt
输入：s = "abcd"
输出："dcbabcd"
```

---

**提示：**

- `0 <= s.length <= 5 * 10^4`
- `s` 仅由小写英文字母组成
